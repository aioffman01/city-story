# 도시이야기 웹 디자인 가이드 (DESIGN.md)

이 문서는 "도시이야기" 웹 애플리케이션에 적용된 디자인 시스템, 컬러 스킴, 레이아웃 및 사용자 경험(UX) 구성 요소를 정리한 문서입니다.

---

## 1. 디자인 컨셉
- **테마**: 프리미엄 글래스모피즘(Glassmorphism) 기반의 다크 모드
- **컨셉 비주얼**: 심해와 밤하늘을 연상케 하는 딥 퍼플(Deep Purple) 배경 위에 선명한 네온 컬러 그라디언트와 은은하게 빛나는 반투명 유리 질감을 결합하여 미래지향적이고 고급스러운 느낌을 부여합니다.

---

## 2. 타이포그래피 (Typography)
- **주 서체**: 구글 웹 폰트 **Outfit**
- **적용 규칙**:
  ```css
  --font-family: 'Outfit', -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, sans-serif;
  ```
- **글꼴 특징**: 둥근 기하학적 요소가 가미되어 다크 테마 내부의 네온 요소들과 부드럽고 세련되게 조화를 이룹니다.

---

## 3. 컬러 시스템 (Color Tokens)
`style.css` 파일에 정의된 주요 CSS 변수 토큰 정보입니다.

| 변수명 | 색상 값 (HEX/RGBA) | 용도 및 설명 |
| :--- | :--- | :--- |
| `--bg-color` | `#0d0b14` | 기본 배경색 (매우 어두운 자줏빛 검정) |
| `--card-bg` | `rgba(255, 255, 255, 0.03)` | 글래스 카드 배경 (반투명 백색) |
| `--card-border` | `rgba(255, 255, 255, 0.08)` | 글래스 카드 기본 경계선 |
| `--card-hover-border` | `rgba(139, 92, 246, 0.4)` | 카드 호버 시 강조되는 보라색 경계선 |
| `--text-primary` | `#f3f1f6` | 주요 텍스트 색상 (연한 은회색) |
| `--text-secondary` | `#9f9bad` | 보조/설명 텍스트 색상 (차분한 회보라색) |
| `--primary-color` | `#8b5cf6` | 메인 테마 보라색 (Violet) |
| `--secondary-color` | `#06b6d4` | 보조 테마 청록색 (Cyan) |
| `--error-color` | `#f87171` | 오류 발생 시 경고 컬러 (Red) |
| `--accent-gradient` | `linear-gradient(135deg, #8b5cf6 0%, #06b6d4 100%)` | 메인 버튼, 랭크 배지, 주요 타이틀 그라디언트 |

- **동적 배경 효과**:
  화면 좌상단(10%, 20%)에 보라색 원형 그라디언트, 우하단(90%, 80%)에 청록색 원형 그라디언트를 블렌딩하여 단조로운 어둠 대신 은은한 깊이감을 주는 오로라 배경 효과를 적용했습니다.

## 4. UI 컴포넌트 설계

### 4.1 글래스 카드 (Glassmorphism Container)
- **스타일 정의**:
  - `backdrop-filter: blur(16px)`를 통한 미려한 유리 반투명 블러 효과
  - 호버 시 약간의 경계선 밝기 변화(`border-color`)와 보라색 은은한 그림자(`box-shadow`) 발생
- **사용처**: 로그인 카드, 회원가입 카드, 메뉴 카드, 도시 정보 카드 등

### 4.2 입력 폼 (Form Controls)
- **스타일 정의**:
  - 비밀번호 외 모든 텍스트, 이메일, 전화번호 입력창 디자인 통일
  - 기본 배경 투명도 유지(`rgba(255,255,255,0.02)`) 및 어두운 경계선
  - 포커스 시 보라색 테두리 광원 효과(`box-shadow: 0 0 0 4px rgba(139, 92, 246, 0.15)`) 제공
  - 왼쪽에 배치된 아이콘 SVG 역시 포커스 시 메인 컬러(`--primary-color`)로 부드럽게 전환

### 4.3 액션 버튼 (Buttons)
- **제출 버튼 (`.btn-submit`)**:
  - 메인 그라디언트 배경 적용
  - 마우스 호버 시 위쪽으로 미세하게 2px 들려 올라가는 입체적 무브먼트 애니메이션 효과(`transform: translateY(-2px)`) 및 광원 그림자 확대 적용

---

## 5. 애니메이션 및 UX 인터랙션
1. **화면 전환 로더 (`.loader`)**:
   - 페이지 진입 시 서서히 페이드아웃 되며 로딩 인디케이터가 회전하는 애니메이션 제공
2. **에러 쉐이크 효과 (`@keyframes shake`)**:
   - 로그인 실패나 폼 미작성 상태 제출 시 경고 창이 좌우로 흔들려 시각적으로 경고를 강력하게 전달
3. **지도 핀 마커 및 팝업 (Leaflet Custom CSS)**:
   - 맵 오버레이에 맞추어 어두운 지도 디자인(CartoDB Dark Matter) 채택
   - 기본 파란색 핀을 대체하는 보라색/청록색 그라디언트 핀과 전용 마커 라벨로 테마 통일성 유지

---

## 6. 관리자 전용 컴포넌트 (Admin Components)

### 6.1 사이트 관리 버튼 (`.btn-admin`)
- **스타일 정의**:
  - 관리자(`admin`) 세션 식별 시 헤더 우측 상단에 동적으로 렌더링됩니다.
  - 보라색(Violet)과 인디고(Indigo) 조합의 투명도 있는 배경 그라디언트(`linear-gradient(135deg, rgba(139, 92, 246, 0.2) 0%, rgba(99, 102, 241, 0.2) 100%)`)를 적용해 고급스러운 차별점을 부여합니다.
  - 마우스 호버 시 배경 불투명도 향상, 보라색 광원 테두리(`rgba(139, 92, 246, 0.8)`) 및 미세한 입체적 움직임(`translateY(-2px)`)을 제공합니다.

### 6.2 관리자 대시보드 통계 카드 및 테이블
- **통계 카드 (`.admin-card`)**:
  - 핵심 지표 텍스트에 백그라운드 클리핑 그라디언트(`linear-gradient(135deg, var(--text-primary) 0%, var(--primary-color) 100%)`)를 적용하여 입체적인 시각성을 극대화합니다.
- **등록 회원 테이블 (`.admin-table`)**:
  - 세련된 다크 테마 표 형태로, 각 로우 호버 시 미세한 밝기 증가 효과(`rgba(255, 255, 255, 0.02)`)를 적용했습니다.
  - 권한 상태는 은은한 녹색 배경의 소프트 배지 스타일(`background: rgba(16, 185, 129, 0.15)`)을 이용해 가독성을 높였습니다.
